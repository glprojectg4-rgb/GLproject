About The System

The Blood & Platelet Donor Management System assists agents, managers, and administrators in managing donors, donations, diseases, centers, and medical alerts.

System Features

Agent Features

  • Manage targeted & universal donations
  • Record recipient information
  • Monitor blood stock
  • Send disease alerts to donors
  • Schedule medical appointments

Collection Manager Features

  • Manage donors
  • Manage CTS centers & sites
  • Communicate with partners & associations
  • Share data between centers

Administrator Features

  • Create user accounts
  • Manage disease list
  • Monitor system activity
